
## DNSMap-C++
DNS-прокси сервер на C++ для динамического маппинга
а IP-адресов.
Является портом оригинального скрипта на Python от ValdikSS, оптимизированным для работы в сетевых окружениях Linux.

## Основные функции

* **Динамический NAT**: Перехватывает `A`-записи DNS и заменяет реальные IP на "фейковые" из заданного пула.
* **Управление Netfilter**: Автоматически прописывает правила `iptables` или `nftables` (DNAT) для прозрачного перенаправления трафика.
* **IPv6 Blocking**: Блокирует `AAAA` и `HTTPS` записи, форсируя использование IPv4.
* **State Recovery**: При запуске считывает текущие правила из системы, восстанавливая таблицу маппинга.
* **Debug Mode**: Подробное логирование сетевых операций и манипуляций с DNS-пакетами.
* **Аргументы**: -a (адрес), -p (порт), -u (апстрим), -r (диапазон).
* **-f**: Запускать в фоновом режиме (не запускать в режиме демона).
* **-d**: Активирует подробный вывод.
## Зависимости

Для сборки и работы необходимы:
* **Библиотека ldns**: `libldns-dev`
* **Компилятор**: GCC с поддержкой C++17
* **Права**: `CAP_NET_ADMIN` для управления фаерволом и `CAP_NET_BIND_SERVICE` для работы на 53 порту.

В Debian/Ubuntu установка зависимостей:
```bash
sudo apt install libldns-dev
```
## Сборка
```bash
make
make BACKEND=nftables # nftables
```
